# Benchmark Filter — Verifiable Salt-Pile Relocation Cases (Strict Gate)

## Inclusion rule (hard gate)
A case is included in the main benchmark only if all fields below are verifiable from credible sources:
1. Initial problem that triggered relocation/transition
2. Timeline (start + end, or clearly in progress with dated record)
3. Cost (or explicit public statement that cost is undisclosed)
4. Downsides/tradeoffs observed during implementation
5. Regulatory/compliance context

If any field is missing, case is excluded from the main benchmark and parked in an evidence backlog.
